Polish KSeF Integration

Krajowy System e-Faktur

Integration with Polish National e-Invoice System (KSeF - Krajowy System e-Faktur).

Features

Send Invoices to KSeF

Easily send customer invoices to the Polish KSeF system with one click.

Track KSeF Numbers

Automatically receive and store KSeF numbers assigned to your invoices.

Status Monitoring

Monitor the status of submitted invoices (Pending, Accepted, Rejected).

FA_VAT Format Support

Automatic generation of invoices in FA_VAT XML format required by KSeF.

Configuration

  1. Go to Accounting → Configuration → KSeF → Configuration
  2. Create a new configuration
  3. Select API Environment (Test or Production)
  4. Enter your KSeF Token
  5. Click "Test Connection" to verify
  6. Enable "Auto-send Invoices" if desired

Usage

Sending Invoices

  1. Create and validate a customer invoice
  2. Click the "Send to KSeF" button in the invoice form
  3. Review the XML preview
  4. Click "Send to KSeF"
  5. Check the KSeF tab for status and assigned number

Requirements

  • Odoo 16.0
  • Python 3.10+
  • cryptography library
  • requests library
  • Valid KSeF Token
